# NLP-Token_NER_SpacyStreamlit

This a NLP App that takes a text and shows 
 + Tokens
 + Parts -of -Speeches
 + Named Entity Recognitions
 
## Tools:
 + Streamlit
 + Spacy
 + ngrok
 
 ## To authenticate ngrok
  + Open and ngrok account
  + Go to ngrok dashboard
  + Copy the Authentication Token
